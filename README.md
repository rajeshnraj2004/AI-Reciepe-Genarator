# 🍳 ChefGPT — AI Recipe Assistant

**ChefGPT** is an AI-powered cooking assistant that helps users **generate recipes, plan meals, and discover dishes** based on available ingredients and preferences.

---

## ✨ Features

* 🧠 **AI Recipe Generation** (from ingredients or prompts)
* 🧾 **Step-by-step cooking instructions**
* 🥗 **Diet filters** (veg/vegan/keto, etc.)
* ⏱️ **Prep & cook time estimates**
* 📋 **Shopping list generator**
* 🔎 **Search & explore recipes**
* 👤 **User profile (optional)**
* 🎨 Clean, modern UI (mobile/web)

---

## 🛠️ Tech Stack

* **Frontend:** React / React Native (Expo)
* **AI Layer:** Custom prompts / local logic *(no API key mode supported)*
* **State & Storage:** Context API / AsyncStorage
* **UI:** Tailwind / custom styles / vector icons

## 🚀 Usage

* Enter ingredients (e.g., *rice, tomato, onion*)
* Choose preferences (diet, cuisine, time)
* Generate recipe
* Follow steps and optionally create a shopping list

---

## 🧠 How It Works

* Inputs → parsed into structured ingredients & constraints
* Rule-based / prompt-driven generator creates:

  * Dish name
  * Ingredients list
  * Step-by-step instructions
  * Time & servings
* Optional: save recipes locally for quick access
